# Proyecto Estructuras Computacionales - Grafos

Aplicación web interactiva desarrollada en Python y Streamlit para el estudio, creación y visualización de grafos.

## Características
*   **Creación Dinámica**: Agrega nodos y aristas con pesos.
*   **Visualización**: Renderizado con `networkx` y `matplotlib`.
*   **Algoritmos**: BFS, DFS, Matrices de Adyacencia e Incidencia.

## Instalación
1.  Clonar el repositorio.
2.  Crear entorno virtual: `python -m venv .venv`
3.  Instalar dependencias: `pip install -r requirements.txt`

## Ejecución
```bash
streamlit run src/app.py
